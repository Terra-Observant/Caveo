# Методология создания гексагональной сетки для оценки риска оползней

## 1. Район исследования

### 1.1 Большой Сочи: общее описание

Городской округ Большой Сочи -- крупнейший курортный город России, расположенный на Черноморском побережье Краснодарского края. Территория простирается вдоль побережья на расстояние более 145 км, что делает его одним из самых протяжённых городских округов мира.

Географические особенности:
- Расположен в горно-лесистой местности
- Значительные перепады высот
- Близость моря и горных рек
- Повышенный риск оползневых процессов

### 1.2 Административное деление

Территория Большого Сочи делится на четыре внутригородских района:

| Район | Код | Расположение | Описание |
|-------|-----|--------------|----------|
| **Адлерский** | 1 | Юго-восточная часть | Включает Адлер, Красную Поляну. Гористый рельеф, высокие перепады высот. |
| **Хостинский** | 2 | Центральная часть побережья | Включает Хосту, Мацесту. Среднегорный рельеф, террасы. |
| **Центральный** | 3 | Центральная часть | Собственно город Сочи. Прибрежная зона, наименьшие перепады. |
| **Лазаревский** | 4 | Северо-западная часть | Наиболее протяжённый район. Гористый рельеф, множество горных рек. |

## 2. Параметры сетки

### 2.1 Обоснование шестиугольной сетки

Для пространственного анализа выбран формат **правильных шестиугольников (hexagon grid)**, что обусловлено следующими преимуществами:

| Преимущество | Описание |
|-------------|----------|
| **Равноудалённые соседи** | У каждого шестиугольника 6 соседей на одинаковом расстоянии. У квадрата -- 4 соседа по стороне и 4 по диагонали (разное расстояние). |
| **Отсутствие диагональных артефактов** | Нет проблемы «углового касания» как у квадратных ячеек. |
| **Лучшее покрытие** | Шестиугольник ближе к кругу, что минимизирует искажения при агрегации данных. |
| **Стабильность при масштабировании** | При агрегации нескольких ячеек в одну более крупную форма остаётся близкой к шестиугольной. |
| **Однородность окрестностей** | Все соседи имеют общую границу одинаковой длины. |

### 2.2 Геометрические параметры

**Система индексации:** H3 (Uber), resolution 9

**Параметры:**
- **Средняя площадь одной ячейки:** ~104 958 м² (~0.105 км²)
- **Средний диаметр ячейки:** ~370 м
- **Идентификатор ячейки:** 15-значное шестнадцатеричное число (например, `891969a6c3fffff`)

H3 resolution 9 выбран как ближайший к исходному требованию (~0.1 км² на ячейку).

### 2.3 Общее количество ячеек

| Показатель | Значение |
|-----------|----------|
| Сгенерировано (до фильтрации) | 751 501 |
| После фильтрации | 34 070 |
| Привязано к районам | 33 555 |
| Не привязано | 515 |

**Критерий включения ячейки:**
- Центроид ячейки находится внутри границ Большого Сочи **ИЛИ**
- Ячейка пересекается с границей минимум на **10%** своей площади

## 3. Система координат

### 3.1 Рабочая система координат

**UTM Zone 37N (EPSG:32637)**

Используется для:
- Построения сетки
- Расчёта площадей
- Расчёта расстояний
- Всех метрических вычислений

**Преимущества:**
- Метрическая система (метры)
- Минимальные искажения в пределах зоны
- Подходит для Сочи (зона 37N, longitude 36°E--42°E)

### 3.2 Система хранения и визуализации

**WGS84 (EPSG:4326)**

Используется для:
- Хранения геоданных (GeoJSON)
- Визуализации на картах
- Совместимости с внешними источниками
- Широты/долготы в градусах

### 3.3 Конвертация

Все данные хранятся в двух проекциях:
- `sochi_hex_grid.geojson` -- UTM (для расчётов)
- `sochi_hex_grid_wgs84.geojson` -- WGS84 (для визуализации)

## 4. Привязка к районам

### 4.1 Метод определения района

Привязка осуществляется по **положению центроида** ячейки:

1. Вычисляется центроид шестиугольника
2. Определяется, в каком районе находится центроид (пространственное соединение `sjoin` с предикатом `within`)
3. Если центроид попадает в границы района -- ячейке присваивается его `district_id` и `district_name`
4. Если центроид вне всех районов -- присваивается `NULL` (серый цвет)

**Обоснование:** Использование центроида обеспечивает однозначную привязку каждой ячейки к одному району, избегая дублирования ячеек на границах.

### 4.2 Цветовая схема

Каждому району назначен уникальный цвет для визуализации:

| Район | Цвет | HEX-код |
|-------|------|---------|
| Адлерский | Синий | `#3498db` |
| Хостинский | Зелёный | `#27ae60` |
| Центральный | Оранжевый | `#e67e22` |
| Лазаревский | Фиолетовый | `#9b59b6` |
| Не определён | Серый | `#95a5a6` |

## 5. Структура атрибутов

Каждая ячейка гексагональной сетки имеет следующие атрибуты:

### 5.1 Идентификация

| Атрибут | Тип | Описание | Пример |
|---------|-----|----------|--------|
| `cell_id` | string | H3 идентификатор ячейки | `891969a6c3fffff` |
| `h3_resolution` | integer | Уровень разрешения H3 | `9` |
| `district_id` | integer | Код района (1--4) или NULL | `1` |
| `district_name` | string | Название района | `Адлерский внутригородской район` |

### 5.2 Координаты

| Атрибут | Тип | Описание | Система |
|---------|-----|----------|---------|
| `centroid_lon` | float | Долгота центроида | WGS84 |
| `centroid_lat` | float | Широта центроида | WGS84 |
| `centroid_x_utm` | float | X центроида (метры) | UTM 37N |
| `centroid_y_utm` | float | Y центроида (метры) | UTM 37N |

### 5.3 Геометрия

| Атрибут | Тип | Описание | Значение |
|---------|-----|----------|----------|
| `area_m2` | float | Площадь ячейки в м² | ~105 000 |
| `area_km2` | float | Площадь ячейки в км² | ~0.105 |
| `coverage_pct` | float | % покрытия территорией | 0--100 |
| `hex_side_m` | float | Примерная длина стороны шестиугольника | ~196 |

### 5.4 Визуализация

| Атрибут | Тип | Описание |
|---------|-----|----------|
| `color` | string | HEX-код цвета по району |

## 8. Проверка корректности

Сетка считается корректной, если выполнены следующие условия:

| № | Критерий | Статус |
|---|----------|--------|
| 1 | Все `cell_id` уникальны | Да |
| 2 | Нет пустых или невалидных геометрий | Да |
| 3 | Площадь каждой ячейки ≈ 105 000 м² (H3 res 9) | Да |
| 4 | Каждая ячейка имеет `district_id` (или NULL для граничных) | Да |
| 5 | Центроиды всех ячеек с `district_id` находятся внутри границ соответствующего района | Да |
| 6 | Сумма площадей ячеек примерно равна площади Большого Сочи | Да |
| 7 | Визуализация показывает четыре района разными цветами | Да |
