# Отчет по формированию набора исторических данных (оползни)

**Источник данных:** Геопортал наземного мониторинга ФГБУ «Гидроспецгеология» (Центр ГМСН).

**Статус:** Сформирован рабочий набор данных (Ground Truth) для обучения модели MVP.

**Файл данных:** `data/processed/historical_landslides.csv`

### 1. Описание структуры данных

В загруженном файле `102 записи об активности оползней` и используются следующие технические поля:

* GEOM: Проекция координат в формате WKT (Point), привязанная к метрической системе (EPSG:3857).

* field_1: Порядковый номер записи в реестре мониторинга.

* field_2: Детальное текстовое описание локации (район г. Сочи, населенный пункт, конкретная улица или километр федеральной трассы А-147/А-148).

* field_3: Широта (Latitude) в десятичных градусах (WGS84) — основной параметр для геопривязки.

* field_4: Долгота (Longitude) в десятичных градусах (WGS84) — основной параметр для геопривязки.

* ngw_id: Уникальный идентификатор объекта в базе данных геопортала.

### 2. Содержание и актуальность

Набор данных содержит  записи об активизации оползневых процессов за период 2023–2024 гг. Это наиболее ценные данные для MVP, так как они соответствуют текущему состоянию рельефа и застройке Большого Сочи. Все точки подтверждены наземными обследованиями специалистов «Гидроспецгеологии».

### 3. Про источник данных

Данные взяты с ГИС-ресурса, который находится по адресу [adler.nextgis.com](https://adler.nextgis.com/resource/0).

Портал также содержит визуализированные слои (интерактивные карты) с границами областей проявления оползневых процессов (ОПОП) по районам Сочи и Адлера.
