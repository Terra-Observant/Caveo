Gitignore и расширенная структура появятся по ходу выполнения
.dvc - настройки/файлы для DVC, пока ничего не кладем
.pipe - пайплайн, тоже не трогаем
data - сюда данные по каталогам(raw - сырые данные(временная папка, потом можно будет удалить), processed - обработанные данные, features - датасет, external - геоданные)
jup_notebooks - код с колаба или jupiter
proj_docs - доки проекта(кроме ридми, ридми в корне репозитория)
src - код проекта(в корень src ничего не кладём, черновик положить в каталог dirty)
training - сюда всё необходимое для обучения модели
На первичные файлы в директориях не обращаем внимание, просто кладем все добро без задней мысли, потом удалим их
